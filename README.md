# Netflix Content Analysis

**Author:** Jasmyne Walker  
**Tools:** R, RStudio, ggplot2, dplyr, readxl  

---

## Project Overview

This project analyzes Netflix movie and TV show data using R.  
The goal is to explore trends in content duration, release years, and ratings through data cleaning and visualization.

---

## Dataset

- Source: Netflix Titles Dataset  
- Format: Excel (.xlsx)  
- Records: Movies and TV shows available on Netflix  

---

## Tools & Libraries

- R  
- RStudio  
- ggplot2  
- dplyr  
- readxl  

---

## Analysis Performed

- Data cleaning and preprocessing  
- Summary statistics  
- Movie duration analysis  
- Distribution visualization  
- Comparison by content type  
- Rating analysis  

---

## Key Visualizations

- Distribution of Netflix movie durations  
- Content growth over time  
- Ratings breakdown  
- Movie vs TV show comparison  

---

## Key Findings

- Most Netflix movies are between 80–120 minutes  
- Content increased significantly after 2010  
- TV-MA and TV-14 are the most common ratings  
- Netflix focuses heavily on feature-length films
